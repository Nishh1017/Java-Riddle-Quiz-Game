# Java-Riddle-Quiz-Game

An interactive console-based riddle game developed in Java using object-oriented programming concepts.

## Features

* Multiple difficulty levels (Easy, Medium, Hard)
* Randomized riddle selection
* Score tracking system
* Replay functionality
* Multiple attempts for each question
* Console-based interactive gameplay

## Concepts Used

* Object-Oriented Programming (OOP)
* Classes and Objects
* Encapsulation
* ArrayLists and Collections Framework
* Loops and Conditional Statements
* Modular Methods
* User Input Handling

## Future Improvements

* Add timer-based gameplay
* Store high scores using file handling
* Add graphical user interface (GUI)
* Expand question database
